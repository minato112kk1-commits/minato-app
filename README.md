# 学習クイズアプリ

AI搭載の学習クイズアプリです。問題を解いて知識を深めよう。

## 機能

- **AI問題生成**: 問題文と正解を入力するだけで、AIが自動的に4択問題を生成
- **成績管理**: クイズの成績を自動記録、統計情報を確認可能
- **カテゴリー分類**: 問題をタイトルで分類して整理
- **マルチプラットフォーム**: モバイルアプリ（iOS/Android）とWebブラウザで利用可能
- **安全なログイン**: Manus OAuth認証で安全にアカウント管理

## 管理者機能

管理者@gmail.com でログインすると以下の機能が利用できます：

- 問題の作成・編集・削除
- AI問題最適化
- 統計情報の確認

## 使い方

1. ログイン - Manus アカウントでログイン
2. クイズを選択 - 学習したい分野を選択
3. 問題に回答 - 4択から正解を選択
4. 成績を確認 - スコアと正答率を確認
